## Codebase Patterns
- Bun 1.3.10: no `Bun.file().textSync()` — use `readFileSync` from `fs` instead
- Tracker parser: legend lines (`- [ ] = ...`) don't contain `/`, use `includes("/")` to filter
- Location pages: always 4 per city (fecal-incontinence, overactive-bladder, pelvic-organ-prolapse, urinary-incontinence)
- Files in ~/.claude/skills/ are sensitive paths; use Bash `cat >` heredoc to write (not Write tool)
- pseo-review-pipeline/ in repo root is gitignored build dir; final destination is ~/.claude/skills/pseo-review/
- Bun.serve() for HTTP servers — no frameworks needed; URL pattern matching on `url.pathname`
- CORS for chrome extensions: match `origin.startsWith("chrome-extension://")` since extension ID varies per install
